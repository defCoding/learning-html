These two tags are known as empty elements because they do not have any words in between the opening and closing tag. Most empty elements only have an opening tag. The space before the / is not needed, but it's added to show that is an empty tag and is a good habit to get into.

Types of Tags

Text Emphasis

Bold is done using <b></b>.
Italics is done using <i></i>.
Underline is done using <u></u>.
Superscript is done using <sup></sup>.
Subscript is done using <sub></sub>.

Strong is done using <strong></strong>. It visually looks the same as bold, but it's used to indicate the content is important. Some readers for the visually impaired will differentiate between strong and bold, so keep that in mind.
Emphasis is done using <em></em>. As with the connection between strong and bold, em is visually identical to italics. However, it carries the meaning that the emphasis will subtly change the meaning of a sentence.

Quotations

Blockquotes can be done using <blockquote></blockquote>. It's used for longer quotes that take up an entire paragraph. Typically, you use <p></p> inside of it. Browsers tend to indent these blockquotes, but won't always do so. Rely on CSS for indentation.
For shorter quotes that sit inside a paragraph, use <q></q>.
Both blockquotes and shorter quotes have a cite attribute with a URL value that will have more information. The cite attribute doesn't have much presentational value, but is there to add semantic information to the quote.

Abbreviations

For an abbr, you can add the full term by using <abbr title=""></abbr>.

Citations and Definitions

According to HTML&CS;S Design and Build Websites, when citing, you can use <cite></cite> to cite work. In HTML5, cite shouldn't be used to cite a person's name, but people usually do anyway. Browsers will render the content of a cite in italics.

The first time you explain some new terminology, you can use the dfn tag to indicate a defining instance is occuring by using <dfn></dfn>. Some browsers show the content in italics, but Safari and Chrome do not. The dfn tag is used to let browsers and web crawlers know that nearby text contains a definition.

Author Details

The address element is fairly niche: it contains the contact details for the author of the page. It can be done using <address></address>.

Changes to Content

The iensrt ins element shows where something has been inserted, and the del element shows where something has been deleted. Insertion uses <ins></ins> and deletion using <del></del>. Typically insertion is underlined, and deletion has a line through it.

The s, or strikethrough, tag can be used to indicate something that is no longer accurate or relevant. It shows as a line through the center, like deletion. Used by doing <s></s>. The difference between s and del is semantical.

Monospace Text

The <tt> element can be used to specify teletype text. However, this tag is now obselete, so should not be used as it could be removed at any time.

Alternatively, the <code> tag can be used to specify computer code, and anything in between the tags (will still have to use special characters for some characters like &am;p) will be treated as code. It is often paired with the <pre> element (preformatted) to preserve line spaces. As with <ol>, <pre> should not be used in <p>.

And while not monospaced, the <var> element is used to indicate variables, and is often presented in italics. y = mx + b.

Lists

There are three types of lists in HTML: ordered, unordered, and definition. Ordered lists are lists where each item in the list is numbered. Unordered lists are lists that begin with a bullet point. Definition lists are made up of a set of terms along with their definitions. One thing to also keep in mind is that list elements should not be stored between <p> tags, because <p> elements contain "phrasing" content, which are inline elements like <strong>. List elements contain "flow" content, which are block elements like <p> and <div>

Ordered Lists

Ordered lists are created using <ol></ol>. Each item in the list is placed between <li> and </li>. Sometimes you may see a type attribute used with ol to specify the type of numbering. But it is better to use CSS for this.

  1. Bring pot of water to a boil.
  2. Sell as lakefront property.
  3. ???
  4. Profit.

Unordered Lists

Unordered lists are created using <ul></ul>. As with ordered lists, each item should be placed in li tags. And as with ordered lists, you may see a type attribute to customize the bullet points, but it's better to use CSS for this.

  • Three feet of hempen rope.
  • Twenty two gallons of goat's milk.
  • Nine square feet of sheet metal.

Definition Lists

Definition lists are created using <dl></dl>. The definition term is stored between <dt> and </dt> tags. The definition for the term is stored in <dd> and </dd> tags. Inside of the dl element, you will usually see pairs of dt and dd elements.

Links

Links allow you to move from one webpage to another.

Types of Links

  • Links from one website to another.
  • Links from one page to another on the same website.
  • Links that open in a new browser window.
  • Links that start up your email program and address a new email to someone.

Writing Links

Links are created using the <a></a> tags. Users can click anywhere in between the tags. To specify the page you want to link to, use the href attribute.
<a href="link goes here">Link text</a>

Linking to Other Sites

When you link to a different website, the value of the href attribute will be the full web address for the site, which is known as an absolute URL.

Example:
<a href="https://www.reddit.com/r/learnprogramming">LearnProgramming Subreddit</a>
LearnProgramming Subreddit

Linking to Other Pages on the Same Site

When you are linking to other pages within the same site, you do not need to use the domain name as the URL. You can instead use the shorthand known as relative URL.

If all the pages of the site are in the same folder, then you can just use the file name for the href attribute.

If they are in different folders, then you will have to follow the directory structure.
The top-level folder is known as the root folder. It contains a file called index.html which is the homepage for the entire site and individual folders such as assets. Web servers are usually set up so that links return to index.html if no file name is specified.

<a href="linkedPage.html">Link to page in the same website.</a>
Link to page in same website.

Email Links

Creating email links is just like creating any other link, except prepend mailto: to the string containing the link, or in this case, email address.

<a href="mailto:thisisnotarealemail@notarealemailhoster.com">Contact Us via Email!</a>
Contact Us via Email!

Opening Links in a New Window

To have a link open in a new window, you can use the target attribute in the <a> element. The value of the attribute should be _blank.

<a href="linkedPage.html" target="_blank">Open new window to page in the same website.</a>
Open new window to page in same website.
